# Flask Counter App

A simple and interactive **Counter Web App** built using **Flask** for the backend, combined with **HTML**, **CSS**, **JavaScript**, and **Tkinter** for desktop GUI support. This app allows users to increment, decrement, and reset a counter, providing both a web-based interface and a standalone desktop application.

## ðŸš€ Getting Started

### Prerequisites

Ensure you have **Python 3.x** installed on your machine. You can download it from [python.org](https://www.python.org/downloads/).

### 1. Clone the Repository

```bash
git clone https://github.com/hayat29/flask_counter_app.git
cd flask_counter_app
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Flask Web App

```bash
python app.py
```

Open your browser and go to [http://127.0.0.1:5000/](http://127.0.0.1:5000/) to interact with the web app.

### 4. Run the Tkinter Desktop App

```bash
python counter_gui.py
```
